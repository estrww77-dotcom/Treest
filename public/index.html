How It Works
Three steps to get started.
1

Download & run RedSea

Download RedSea.exe from the releases page and run it on your Windows PC. No installation required.

2

Configure Steam

Click Patch Steam inside the app to apply the necessary configuration to your Steam directory.

3

Add a game via Game Hub

Search for a game by name or AppID, click it, and RedSea sets it up automatically. You can also use this website's Game Hub to browse and add games manually.

4

Restart Steam

Click Restart Steam in the app (or use the button on screen). Changes take effect after Steam restarts. That's it.
